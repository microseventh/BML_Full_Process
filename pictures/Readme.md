### 本文档用于收集文档对应的图片，具体命名格式如下

1. 每一章每一节为一个文件夹，文件夹命名为「章序号节序号」
2. 每一个文件夹的图片命名，格式为「章序号节序号-分类标签-序号」
3. 根据实际情况，分类标签有所不同，简要列举如下

--- 

| 分类标签    | 标签解释 |
|:-------:|:----:|
| samples | 举例   |
| flow    | 流程   |
| summary | 总结   |

--- 

4. 序号对应为：从01开始，逐步递增1，最高到99。
5. 序号可能会断开，因为需要保留一部分序号用于补充或修改。
